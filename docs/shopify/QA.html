问题列表

Q:在后台添加的商品信息，会写入到本地的文件吗？

1. Q:在后台添加的商品信息，会写入到本地的文件吗？

A:在 Shopify 中，当你在后台（例如你提到的链接）添加或删除产品时，这些数据不会写入主题代码中的任何文件。它们被存储在 Shopify 的云端数据库中，并通过 Shopify 的 API 进行管理。

2. Q:如何添加商品的第二张，甚至第三张图片？

A:在 Shopify 后台，媒体文件部分直接添加，选中上传的图片资源即可。这个同样是存储在 Shopify 的云端数据库中。在代码中是直接调用 product.media[]来进行图片悬浮切换操作的。

3. Q:在在线网站修改的一些参数，为什么不会修改本地的？

A: Shopify 并不会把设置变更写入你本地文件系统。 所有设置变更（包括 checkbox 选中状态）都存储在 Shopify 云端的 settings_data.json 中，而不是你的本地文件里。

4. Q:如果我想让本地同步用户设置，怎么办？

可以使用以下命令将线上配置文件拉取下来：

shopify theme pull

这个命令会把在线商店中激活主题的所有配置（包括 settings_data.json）拉取下来 写入到你的本地项目中（特别是 config/ 文件夹）

🛑 注意：这个是覆盖行为，如果你本地改了别的东西没 push，可能被覆盖。
